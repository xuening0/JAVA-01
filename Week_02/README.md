串行GC与并行GC效果差不多, 并行GC的youngGC次数> 串行GC, 导致并行GC时间变长；
cmsGC< 并行GC 并行GC的youngGC次数> cmsGC, 因为两者youngGC都是ParallelGC,cmsGC在oldGC时退化成串行GC   
G1GC pause时间 < 三者,内存越大越明显, 产生的对象也越来越多.

注意cmsGC退化成serialGC, 使用G1GC内存要尽可能大
